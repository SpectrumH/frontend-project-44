Clone the repository with the command
(html)
`git clone https://github.com/SpectrumH/frontend-project-44`
or (ssh)
`git clone git@github.com:SpectrumH/fronend-project-44.git`
Install dependencies
`npm ci`

Publish the project locally
`make publish`

Install using npm
`npm link`
